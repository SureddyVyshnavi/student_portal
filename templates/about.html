🎓 About Student Portal

Student Portal is a centralized web application designed for colleges to manage student records, book borrowing, registrations, and institutional information — all in one place. Built with usability and security in mind, it brings together students, teachers, and administrators in a seamless digital environment.

🚀 Key Features

  • 🔐 Secure login and user registration system
  • 📝 Register and manage student data with course, contact, and profile details
  • 📚 Browse books, add to cart, and view current selections
  • 👨‍🎓 View all registered students in a searchable list
  • 🏫 Dedicated sections for college information and contact form with message handling

👥 Who Can Use the Portal?

This platform is ideal for college students, faculty members, and administrators who wish to streamline student and academic data handling using a paperless and digital solution.

🧰 Technologies Used

  • 🐍 Python with Flask for the backend logic
  • 🗃️ PostgreSQL as a robust relational database
  • 🎨 HTML & CSS for frontend design and layout
  • 🛰️ Deployed using IIS or Render for web hosting

🎯 Our Mission

We aim to simplify how educational institutions handle student data and improve the overall academic experience by providing an intuitive, secure, and efficient digital solution.
